Command ReferenceÔÉÅ

See below for available options for the various commands.

  • Common Options
    • Common Options
  • Server
    • Service List : cmdbox -m server -c list <Option>
    • Service Launch : cmdbox -m server -c start <Option>
    • Service Stops : cmdbox -m server -c stop <Option>
  • Client
    • client ( Copy File ) : cmdbox -m client -c file_copy <Option>
    • client ( Download File ) : cmdbox -m client -c file_download <Option>
    • client ( List File ) : cmdbox -m client -c file_list <Option>
    • client ( Create Folder ) : cmdbox -m client -c file_mkdir <Option>
    • client ( Move File ) : cmdbox -m client -c file_move <Option>
    • client ( Delete File ) : cmdbox -m client -c file_remove <Option>
    • client ( Delete Folder ) : cmdbox -m client -c file_rmdir <Option>
    • client ( Upload File ) : cmdbox -m client -c file_upload <Option>
    • client ( Server Info ) : cmdbox -m client -c server_info <Option>
  • GUI
    • Management screen startup : cmdbox -m gui -c start <Option>
  • Web
    • Self-signed server certificate generation : cmdbox -m web -c gencert <Option>
    • Add Group : cmdbox -m web -c group_add <Option>
    • Delete Group : cmdbox -m web -c group_del <Option>
    • Edit Group : cmdbox -m web -c group_edit <Option>
    • List Group : cmdbox -m web -c group_list <Option>
    • Web Service Launch : cmdbox -m web -c start <Option>
    • Web Service Stops : cmdbox -m web -c stop <Option>
    • Add User : cmdbox -m web -c user_add <Option>
    • Delete User : cmdbox -m web -c user_del <Option>
    • Edit User : cmdbox -m web -c user_edit <Option>
    • List User : cmdbox -m web -c user_list <Option>
